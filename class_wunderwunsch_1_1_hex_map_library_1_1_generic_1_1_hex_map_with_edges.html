Wunderwunsch.HexMapLibrary.Generic.HexMapWithEdges< T, E > Class Template Reference

An Instance defines a specific map made out of hexagonal tiles. Stores each valid map tile and edge which can be accessed by their position or index; Provides the same methods as the Hex Class but removes all results which are out of the map bounds. This base implementation only works on contigious, convex, non-wrapping maps without holes. Methods which potentially need to be abjusted for other map types are marked as virtual. See HexMapRectangularHorizontalPeriodic for a implementation for maps which wraps around on the offset/cartesian X-Axis. More...
