Detailed Description

Add this as custom IEqualityComparer to any Dictionary using Vector3Int as key, this is unfortunately necessary to prevent boxing (and therefore memory allocations) as Unity omitted to implement the IEquatable<Vector3Int> interface on Vector3Int
