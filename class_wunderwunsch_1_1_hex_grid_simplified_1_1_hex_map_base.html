Detailed Description

Base class of HexMap - which for most use cases is simpler to work with. An Instance defines a specific map made out of hexagonal tiles. Stores each valid map tile position and edge position and their index. Does not store the tiles and edge objects itself, use HexMap instead if you want that. Provides the same methods as the Hex Class but removes all results which are out of the map bounds. This base implementation only works on contigious, convex, non-wrapping maps without holes. Methods which potentially need to be abjusted for other map types are marked as virtual.
