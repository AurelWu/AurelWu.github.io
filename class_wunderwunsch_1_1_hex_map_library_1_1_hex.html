Detailed Description

Contains many helper methods facilitating working with a hexagonal coordinate system. Assumes the grid being on the (infinite) XZ-Plane.
That means no wrapping logic or removal of invalid tiles - this functionality is provided by HexMapBase and HexMap.

Even when working with a finite map, the methods of this class can be useful for interim results.
